# bfg-macro

This is a simple macro software for the game [Growtopia](https://growtopiagame.com/) using [pyautogui](https://pypi.org/project/PyAutoGUI/). 
In order to use it in game you must be zoomed in all the way and have your deployable at the slot next to your fist.
You have to input your farming position, your farming range and the hit points of your deployable correctly.
